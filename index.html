Overview of Bambi command-line tools

Bambi is a collection of tools to generate or process BAM/SAM files using the HTSLib 'C' library.

It currently includes i2b, decode, select, chrsplit, and read2tags

The Bambi command-line tools are packaged as a single executable and are run as follows:

bambi command [options]
